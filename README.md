# README #

### What is this repository for? ###

* What's new in Java 9, 10, 11
* Migration to latest Java LTS release
* Examples using JShell

### Build

To create a PDF:
`./build-pdf.sh`

To create a self-contained HTML5 page:
`./build-html.sh`
